VIM 快捷键和简单的使用方法

小伟

2019-10-24

Linux / Vim

推荐常用命令：

  • . 重复上一次的操作
  • *光标放在单词上直接查找该单词
  • u 恢复上一步的操作

模式切换

  • i进入插入模式
  • Esc 或<C-[> 退回命令模式
  • 临时进入命令模式或插入模式

插入模式

  • a 在光标后面插入
  • A 在当前行的最后面插入
  • i 在光标当前位置插入

查找模式

  • /内容 直接查找你要的内容
  • n 跳转到下一个查找匹配的内容
  • N 跳转的上一个查找匹配的内容
  • *光标放在单词上直接查找该单词

编辑模式

光标移动
  • j 向下移动
  • k 向上移动
  • h 向左移动
  • l 向右移动
  • n 向后移动n个字符
  • n 向下移动n行字符
  • G 移动到最后一行
  • nG 移动到n行
  • gg 移动到首行
  • Ctrl+f 向下翻页
  • Ctrl+b 向上翻页
  • Ctrl+u 向上翻半页
  • Ctrl+d 向下翻半页
  • f字符 将光标定位到向后查找的字符上
  • F字符 将光标定位到向前查找的字符上
    删除
  • dw 先后删除一个单词
  • db 先前删除一个单词
  • dd 删除当前行
  • ndd 向下删除n行
  • d1G 删除光标所在到第一行的所有数据
  • dG 删除光标所在到最后一行的所有数据
  • d$ 删除游标所在处，到该行的最后一个字符
  • d0 那个是数字的 0 ，删除游标所在处，到该行的最前面一个字符
    复制
  • yy 复制游标所在的那一行(常用)
  • nyy n 为数字。复制光标所在的向下 n 行，例如 20yy 则是复制 20 行(常用)
  • y1G 复制游标所在行到第一行的所有数据
  • yG 复制游标所在行到最后一行的所有数据
  • y0 复制光标所在的那个字符到该行行首的所有数据
  • y$ 复制光标所在的那个字符到该行行尾的所有数据
